task: Generating PDF documents populated with user data from prepared PDF templates.

Short description

Wrapper for TCPDF class for creating documents from original PDF template, 
by populating it with user data.
It has features for printing "data grids" and repeating data on the page 
(for example business cards printing),
drawing bar codes, QR-codes, polygons, images from source picture files etc.
